# WeaTheraPy

## Coded in Python

### A simple Weather Checking CLI-based App

#### Features:

1. Automatic User Location Fetching
2. Personalized Experience
3. 5-Day Weather Forecasting
4. Easy Setup
5. Well-structured Code

#### How to use it?

1. Go over to: [OpenWeatherMap](https://openweathermap.org/) and generate an API Key.
2. Run the .exe (execution) file directly.
3. Enter the API Key.
4. Enjoy :)
